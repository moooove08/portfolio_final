Follow Your Bet

Personal sports analytics tracker

A local-only personal tracking app for saving personal bet records, reviewing win/loss results, tracking profit/ROI and visualizing performance through dashboard screens.

Product flow

What the app demonstrates

  • Home dashboard with profit, ROI, recent records and donut chart
  • Add-record flow with validation, date picker and keyboard handling
  • Full list grouped by month/year and sorted by recency
  • Analytics screen with filters, category breakdown and top records
  • Fully local storage with UserDefaults and Codable

Technical focus

Implementation highlights

  • Structured UI, domain services and repositories with lightweight Clean Architecture
  • Built chart-based analytics screens using the Charts framework
  • Created a consistent dark visual identity and custom tab bar
  • Designed for personal tracking only — no bookmaker connection and no real-money transactions

Portfolio note

Public GitHub repository is available. The README explicitly presents it as a personal tracking app and notes that it does not place bets, connect to bookmakers or process real-money transactions.
