Open-source UIKit project

Habbitator

Habit tracking app with Core Data and MVVM

An iOS habit-building app where users choose three habits for the day, complete them, review calendar history and track progress/streaks.

Product flow

What the app demonstrates

  • Create three daily habits and keep or replace them on a new day
  • Mark tasks as completed and track successful-day streaks
  • Review progress and completion history in a calendar
  • Switch between multiple visual themes
  • Store habits and daily completion records locally with Core Data

Technical focus

Implementation highlights

  • Split the app into App, Core, Domain, Data and Presentation layers
  • Controllers receive prepared screen data from ViewModels instead of reading Core Data directly
  • Repository protocol separates domain logic from persistence implementation
  • Demonstrates a stronger architecture than a purely screen-based demo
