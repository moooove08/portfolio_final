Forest Flicks

Motion-controlled iOS archery game

A landscape iOS archery game where the iPhone acts as a motion controller and a forward thrust fires an arrow at the target.

Product flow

What the app demonstrates

  • External-screen oriented gameplay concept with iPhone motion controls
  • CoreMotion shot gesture detection at high frequency
  • Store and arrow selection with persistent stock/balance
  • Target scoring zones with multipliers
  • Landscape game experience with custom visual assets

Technical focus

Implementation highlights

  • Implemented motion-based shooting using acceleration and rotation thresholds
  • Separated SpriteKit layout, gameplay, scoring and gesture detection responsibilities
  • Persisted progress through local storage while keeping the game lightweight
  • Designed a portfolio-friendly game concept that stands out from standard CRUD apps

Portfolio note

Public GitHub repository is available. The README explains the CoreMotion shooting mechanics, scoring formula, local progress persistence and game module structure.
