Checklist app with templates and themes

A compact UIKit checklist app for creating personal lists, reusable templates, custom icons/colors, visual themes and archiving completed lists.

Product flow

What the app demonstrates

  • Create, edit and delete checklists
  • Add, edit, check and delete list items
  • Archive completed lists automatically
  • Create lists from built-in and user-saved templates
  • Customize list icons, accent colors and visual themes

Technical focus

Implementation highlights

  • Built table-driven editing and archive flows
  • Persisted lists, templates and theme state with UserDefaults + Codable
  • Created a lightweight design system for list colors and visual styles
  • Demonstrates clean UIKit fundamentals and local state handling
