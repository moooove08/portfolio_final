Gaming club booking and catalogue prototype

A neon-style entertainment venue app prototype built around a game club experience: promo lobby, game space booking, console categories, curated game lists and tab-based navigation.

Product flow

What the app demonstrates

  • Branded loading and home/lobby screens with promotional cards
  • Game space booking screen with date/time-style booking context
  • Game catalogue grouped by categories such as arcade, pinball, manual games and consoles
  • Platform sections for Nintendo, PlayStation, Xbox, Retro and VR-style game booths
  • Tab navigation that simulates a full commercial mobile product

Technical focus

Implementation highlights

  • Built a polished entertainment app flow focused on visual identity and user journey
  • Designed content and navigation modules that can support remote catalogues, user sessions and live promotional content
  • Designed screens that can support booking, membership, QR pass, loyalty or player-profile modules
  • Kept the presentation strong while clearly separating prototype presentation from external service integrations

Portfolio note

Source code is not public. This case study focuses on a commercial-style entertainment product with strong UI identity, content structure, booking-oriented flow and interaction states. In a production setup, the same direction can be supported by Firebase services, REST APIs, user profiles, loyalty flows and remote content delivery.
