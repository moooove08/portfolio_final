# Vlad Kuzmenko — iOS Portfolio

Static portfolio site for GitHub Pages.

## What is included

- `index.html` — main portfolio page
- `projects/*.html` — detailed case study pages
- `assets/projects/*` — optimized screenshots, covers and mp4 demos
- `assets/cv/*` — CV files
- `linkedin-profile-texts.md` — reusable LinkedIn text blocks

- Book Sport Transport — Product-like iOS Prototype
- BC Game Space — Commercial-style iOS App Demo
- Habbitator — Open-source UIKit project
- Forest Flicks — Open-source SpriteKit/CoreMotion project
- Follow Your Bet — Open-source UIKit project
- Daily List — Open-source UIKit project
- Chickens & Beast — Open-source UIKit/XIB game project

## Deploy to GitHub Pages

1. Create or open a repository, for example `moooove08.github.io` or `ios-portfolio`.
2. Copy all files from this folder into the repository root.
3. Push to GitHub.
4. In repository Settings → Pages, choose the `main` branch and `/root` folder.

The site has no build step and can be opened locally by double-clicking `index.html`.
